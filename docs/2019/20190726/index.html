Is Life Always This Hard?

Written by MeoWoo with ♥ on July 26, 2019 in

臭大便

生活是大便，

每天你等它，

不来你焦虑，

拉完又嫌弃，

来去冲冲啦。


绿豆汤

绿豆绿豆汤，

为啥红色呀，

你妈说你傻，

叫你自个查。

做规划

你说二狗他，

啥也不牵挂，

催他做规划，

他说别急嘛。

打油诗

你没点眼力劲啊？
